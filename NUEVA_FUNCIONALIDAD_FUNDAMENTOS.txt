═══════════════════════════════════════════════════════════════════

🎯 FUNDAMENTOS DISPONIBLES:

1. 🏐 Todos los fundamentos
   - Plan de entrenamiento completo con todos los fundamentos
   - Desarrollo integral de habilidades

2. 🏐 Saque
   - Técnica de saque bajo y alto
   - Precisión en zonas específicas
   - Diferentes tipos de efectos
   - Competencias de precisión

3. 🤲 Recepción
   - Pase de antebrazo
   - Recepción de saque
   - Formación W
   - Desplazamientos y posicionamiento
   - Recepción a zona específica

4. 🙌 Armado/Levantada
   - Pase de dedos
   - Posicionamiento de armador
   - Armado desde diferentes posiciones
   - Coordinación recepción-armado-ataque
   - Técnica de muñecas

5. ⚡ Ataque/Remate
   - Aproximación y timing
   - Remate desde diferentes zonas
   - Coordinación brazos-salto
   - Combinaciones de ataque
   - Remate con oposición

6. 🛡️ Bloqueo
   - Técnica de bloqueo individual
   - Desplazamiento lateral en red
   - Bloqueo doble coordinado
   - Lectura del atacante
   - Transición defensa

7. 🏃 Defensa
   - Posición baja y movimientos
   - Defensa de ataques
   - Plancha y salvadas
   - Sistemas defensivos
   - Transición contraataque

═══════════════════════════════════════════════════════════════════

💡 CÓMO FUNCIONA:

1. FILTRADO INTELIGENTE
   ✓ El sistema analiza todos los ejercicios de la base de datos
   ✓ Identifica ejercicios relacionados con el fundamento elegido
   ✓ Usa palabras clave específicas para cada fundamento
   ✓ Filtra y selecciona solo ejercicios relevantes

2. EJERCICIOS GENÉRICOS
   ✓ Si no hay suficientes ejercicios en la base de datos
   ✓ Genera automáticamente ejercicios específicos del fundamento
   ✓ Incluye calentamiento especializado
   ✓ Progresión técnica adecuada

3. PLAN PERSONALIZADO
   ✓ El plan muestra el fundamento seleccionado
   ✓ Todos los ejercicios están enfocados
   ✓ Respeta el límite de 120 minutos
   ✓ Se adapta a la cantidad de asistentes
