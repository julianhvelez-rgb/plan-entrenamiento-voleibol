═══════════════════════════════════════════════════════════════════════════

   🔷 TABLETA ANDROID O iPAD:
      └─→ Usa: app_tableta.py (versión web)
      └─→ Ejecuta: INICIAR_TABLETA.bat

   🔷 TABLETA WINDOWS (Surface, etc.):
      └─→ Usa: app_tableta_standalone.py (esta versión)
      └─→ Ejecuta: INICIAR_TABLETA_STANDALONE.bat
      └─→ O compila: COMPILAR_TABLETA.bat (para crear .exe)

   🔷 PC/LAPTOP con PANTALLA TÁCTIL:
      └─→ Usa: app_tableta_standalone.py (esta versión)
      └─→ Mejor experiencia como app independiente

═══════════════════════════════════════════════════════════════════════════
🚀 INICIO RÁPIDO - 2 OPCIONES
═══════════════════════════════════════════════════════════════════════════

   OPCIÓN 1 - Ejecutar directamente (más rápido):
   ───────────────────────────────────────────────
   1. Doble clic: INICIAR_TABLETA_STANDALONE.bat
   2. Se abre la aplicación en ventana propia
   3. ¡Úsala! 🎉

   (Requiere Python instalado)


   OPCIÓN 2 - Crear .exe portable (recomendado):
   ───────────────────────────────────────────────
   1. Doble clic: COMPILAR_TABLETA.bat
   2. Espera 5-10 minutos (se instalará todo automáticamente)
   3. Busca: dist\EntrenamientoVoleibol_Tableta.exe
   4. Copia el .exe donde quieras (USB, tablet, otra PC)
   5. Ejecuta el .exe - ¡Funciona sin instalar NADA! 🎉

🆚 DIFERENCIA CON LA VERSIÓN WEB
═══════════════════════════════════════════════════════════════════════════

   VERSIÓN WEB (app_tableta.py):
   • Abre Chrome/Safari/Edge
   • Funciona en Android, iOS, Windows
   • Necesita servidor + WiFi
   • Múltiples dispositivos

   VERSIÓN STANDALONE (app_tableta_standalone.py):
   • NO necesita navegador
   • Solo Windows
   • Ventana independiente
   • Se crea .exe portable
   • ⚡ Más rápida y directa
