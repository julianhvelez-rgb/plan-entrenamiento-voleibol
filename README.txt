   🏐 SISTEMA DE ENTRENAMIENTO DE VOLEIBOL 🏐
═══════════════════════════════════════════════════════════════

📋 DESCRIPCIÓN:
--------------
Aplicación independiente para planificar entrenamientos de voleibol
para niñas. Incluye visualizaciones gráficas de la cancha y ejercicios.

🚀 CÓMO USAR:
-------------
1. Busca el archivo: "EntrenamientoVoleibol.exe" en la carpeta "dist"

2. Haz doble clic en "EntrenamientoVoleibol.exe" para iniciar

3. NO necesitas tener Python instalado
   NO necesitas conexión a Internet
   
4. La aplicación funciona completamente offline


📂 UBICACIÓN DEL EJECUTABLE:
-----------------------------
dist\EntrenamientoVoleibol.exe

Tamaño: ~9.4 MB
Tipo: Aplicación Windows (.exe)


✨ CARACTERÍSTICAS:
------------------
✓ Selección de rangos de edad (8-10, 11-13, 14-16, 17+ años)
✓ Registro de asistentes del día
✓ 5 Etapas de entrenamiento:
  • Preparación General
  • Preparación Específica
  • Precompetitiva
  • Competitiva
  • Transición

✓ Base de datos completa de ejercicios
✓ Visualizaciones gráficas:
  • Cancha completa con zonas numeradas
  • Formaciones de recepción (W y 3-2-1)
  • Diagramas de ejercicios en parejas
  • Ejercicios en círculo
  • Cadena recepción-colocación-remate
  • Bloqueo y defensa
  • Organización por estaciones

✓ Planes personalizados
✓ Guardar planes en archivos .txt
✓ Recomendaciones según cantidad de asistentes


🎯 OPCIONES DEL MENÚ:
---------------------
1. Crear Plan de Entrenamiento del Día
   → Plan rápido con toda la información

2. Ver Base de Ejercicios por Etapa
   → Explorar ejercicios disponibles

3. Generar Plan Personalizado
   → Seleccionar ejercicios específicos

4. Ver Diagramas de Cancha y Ejercicios
   → Visualizaciones gráficas detalladas

5. Salir
   → Cerrar la aplicación


💾 PORTABILIDAD:
----------------
Puedes copiar el archivo "EntrenamientoVoleibol.exe" a:
- Una memoria USB
- Otro computador
- El escritorio
- Cualquier carpeta

Y funcionará sin problemas, sin necesidad de instalación.


📝 PLANES GUARDADOS:
--------------------
Los planes de entrenamiento se guardan automáticamente como:
"plan_entrenamiento_AAAAMMDD_HHMMSS.txt"

En la misma carpeta donde ejecutes el programa.


⚠️ NOTAS IMPORTANTES:
---------------------
• Primera ejecución puede tardar unos segundos en abrir
• Windows Defender puede mostrar advertencia (es normal)
• El archivo es seguro, creado con PyInstaller
• Si Windows bloquea, haz clic en "Más información" → "Ejecutar de todas formas"


🔧 REQUISITOS DEL SISTEMA:
--------------------------
• Windows 7 o superior
• ~10 MB de espacio en disco
• No requiere conexión a Internet
• No requiere Python instalado


📧 SOPORTE:
-----------
Para soporte o preguntas sobre el uso de la aplicación,
consulta el archivo ejerciciosvoly.py (código fuente).
