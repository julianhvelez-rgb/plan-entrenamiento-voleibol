✨ NUEVA VERSIÓN WEB OPTIMIZADA PARA TABLETAS ✨

Esta es una versión web de la aplicación de entrenamiento de voleibol,
diseñada específicamente para usar en tabletas (Android, iOS, Windows).

═══════════════════════════════════════════════════════════════════════════
📦 ARCHIVOS NUEVOS CREADOS:
═══════════════════════════════════════════════════════════════════════════

  📄 app_tableta.py                  - Servidor Flask principal
  📄 INICIAR_TABLETA.bat            - Ejecutar para iniciar el servidor
  📄 INSTRUCCIONES_TABLETA.txt      - Guía completa de uso
  📄 requirements_tableta.txt       - Dependencias necesarias
  
  📁 templates/
     └── index.html                  - Interfaz web principal
  
  📁 static/
     ├── css/
     │   └── styles.css              - Estilos responsive
     └── js/
         └── app.js                  - JavaScript interactivo

═══════════════════════════════════════════════════════════════════════════
🚀 INICIO RÁPIDO:
═══════════════════════════════════════════════════════════════════════════

  1️⃣  Haz doble clic en: INICIAR_TABLETA.bat
  
  2️⃣  Anota la dirección IP que aparece (ej: http://192.168.1.100:5000)
  
  3️⃣  En tu tableta, abre el navegador y escribe esa dirección
  
  4️⃣  ¡Listo! Ya puedes usar la aplicación en tu tableta 🎉

═══════════════════════════════════════════════════════════════════════════
⚙️ INSTALACIÓN MANUAL (si el .bat no funciona):
═══════════════════════════════════════════════════════════════════════════

  1. Instalar Flask:
     pip install -r requirements_tableta.txt
  
  2. Iniciar servidor:
     python app_tableta.py
  
  3. Acceder desde tableta:
     http://[TU-IP]:5000

📱 CÓMO AGREGAR A LA PANTALLA DE INICIO:
═══════════════════════════════════════════════════════════════════════════

  ANDROID/CHROME:
  1. Abre el menú (⋮)
  2. "Agregar a pantalla de inicio"
  3. Ahora tienes un ícono directo a la app

  iPAD/SAFARI:
  1. Toca el botón compartir (□↑)
  2. "Añadir a pantalla de inicio"
  3. Ahora tienes un ícono directo a la app

  ➜ La app se verá como una aplicación nativa
