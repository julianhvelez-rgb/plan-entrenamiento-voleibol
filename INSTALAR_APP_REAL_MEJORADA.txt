═══════════════════════════════════════════════════════════════════════════
❌ PASO 1: ELIMINAR EL ACCESO DIRECTO VIEJO
═══════════════════════════════════════════════════════════════════════════

Antes de instalar la versión mejorada, elimina el acceso directo viejo:

1. En tu tableta Android, busca el ícono viejo en la pantalla de inicio

2. Mantén presionado el ícono hasta que aparezcan opciones

3. Arrastra a "Eliminar" o toca "Desinstalar"

4. Confirma

✅ El acceso directo viejo ha sido eliminado

═══════════════════════════════════════════════════════════════════════════
🔄 PASO 2: LIMPIAR CACHÉ DE CHROME
═══════════════════════════════════════════════════════════════════════════

Para asegurarnos de que Chrome detecte la nueva configuración:

1. Abre Chrome en tu tableta

2. Toca el menú (⋮) → Configuración

3. Privacidad y seguridad → Borrar datos de navegación

4. Selecciona:
   ☑️ Imágenes y archivos en caché
   ☑️ Cookies y datos de sitios

5. Toca "Borrar datos"

6. Cierra Chrome completamente (desde el multitarea)

✅ Caché limpiado

═══════════════════════════════════════════════════════════════════════════
✅ PASO 3: INSTALAR LA APP REAL MEJORADA
═══════════════════════════════════════════════════════════════════════════

Ahora vamos a instalar la versión PWA mejorada:

1. Abre Chrome en tu tableta

2. Ve a: http://10.0.0.156:5000

3. Espera a que cargue COMPLETAMENTE (importante)

4. Ahora hay DOS formas de instalar:

   FORMA A - Banner de instalación (recomendada):
   ─────────────────────────────────────────────
   • Chrome mostrará un banner en la parte inferior:
     "Agregar Voleibol Pro a la pantalla de inicio"
   
   • Toca el banner
   
   • Verás un popup con el ícono de la app y el nombre
   
   • Toca "Agregar" o "Instalar"
   
   • ¡La app se instalará! 🎉


   FORMA B - Desde el menú (si no aparece el banner):
   ────────────────────────────────────────────────────
   • Toca el menú (⋮) en la esquina superior derecha
   
   • Busca la opción: "Instalar app" o "Agregar a inicio"
     (Debería tener el ícono 🏐 al lado)
   
   • Toca esa opción
   
   • Confirma "Instalar"
   
   • ¡La app se instalará! 🎉

5. Cierra Chrome

6. Ve a tu pantalla de inicio

7. Verás el nuevo ícono con logo de voleibol 🏐

8. ¡TOCA EL ÍCONO PARA ABRIR LA APP!

═══════════════════════════════════════════════════════════════════════════
🔍 PASO 4: VERIFICAR QUE ES UNA APP REAL
═══════════════════════════════════════════════════════════════════════════

Cuando tocas el ícono y se abre, debes verificar:

✅ SE ABRE EN PANTALLA COMPLETA
   → NO se ve la barra de Chrome arriba
   → NO se ve la URL
   → NO se ven los botones de navegación
   → Solo ves tu aplicación

✅ TIENE SU PROPIA VENTANA
   → En el multitarea (apps recientes) aparece como app separada
   → Con su propio ícono de voleibol 🏐

✅ SE COMPORTA COMO APP NATIVA
   → Puedes cambiar entre apps normalmente
   → Aparece en el listado de aplicaciones
   → Se puede "cerrar" como cualquier app

═══════════════════════════════════════════════════════════════════════════
❓ SI TODAVÍA SE INSTALA COMO ACCESO DIRECTO
═══════════════════════════════════════════════════════════════════════════

Si después de seguir todos los pasos aún se instala como acceso directo:

SOLUCIÓN 1 - Actualizar Chrome:
────────────────────────────────
1. Play Store → Buscar "Chrome"
2. Si hay actualización disponible, actualizar
3. Reiniciar el dispositivo
4. Intentar instalar de nuevo

SOLUCIÓN 2 - Verificar configuración de Chrome:
───────────────────────────────────────────────
1. Chrome → Configuración
2. Configuración del sitio
3. Buscar: localhost o 10.0.0.156
4. Eliminar datos de ese sitio
5. Volver a intentar

SOLUCIÓN 3 - Forzar instalación PWA:
────────────────────────────────────
1. Con la página abierta en Chrome
2. Menú (⋮) → "Agregar a inicio"
3. En el popup que aparece:
   • Cambia el nombre a: "Voleibol Pro"
   • NO toques "Agregar" todavía
4. Espera 3-5 segundos
5. Chrome debería detectar que es PWA y cambiar el popup
6. Ahora sí toca "Agregar" o "Instalar"
